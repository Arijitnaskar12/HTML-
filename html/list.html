Block Elements

Defination:

Block elements are the elements which always start with a new line and that also takes the width of the screen.

  1. paragraph tag
  2. heading tag
  3. Div tag
  4. Ul tag

Inline Elements

Defination:

Inline elements are the elements which always start with where the previous elements have ended.

  • audio tag
  • video tag
  • span tag
  • input tag
  • label tag
